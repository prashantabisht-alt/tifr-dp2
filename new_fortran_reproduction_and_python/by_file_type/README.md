# Browse by File Type

These folders group the TCRW reproduction files by extension for easier GitHub
browsing.

- `01_python/`: Python reproductions, exact-matrix checks, and plotting helpers.
- `02_fortran/`: Fortran Monte Carlo drivers plus shared step/random kernels.
- `03_gnuplot/`: gnuplot scripts used by `run.sh`.
- `04_text_data/`: simulation summaries, trajectories, currents, and exact data.
- `05_figures/`: generated `.png` and `.pdf` outputs.
- `06_notebooks/`: Jupyter audit notebooks.
- `07_reports/`: Markdown audit reports and convention notes.
- `08_logs/`: run logs.
- `09_shell/`: shell driver scripts.

These are links, not duplicate files. The runnable source of truth remains the
parent `new_fortran_reproduction_and_python/` folder.
